# Descrição do Programa

Este programa gerencia informações de trabalhadores, seus departamentos e contratos por hora. Ele permite:

- Cadastrar um trabalhador com nome, nível, salário base e departamento.
- Registrar múltiplos contratos de trabalho por hora, incluindo data, valor por hora e duração em horas.
- Calcular a renda mensal do trabalhador para um dado mês e ano, somando o salário base e o valor total dos contratos realizados naquele período.

O programa utiliza classes para representar trabalhadores, contratos, departamentos e níveis profissionais, aplicando conceitos de orientação a objetos em Java.
